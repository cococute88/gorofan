# 개인용 AI 캐릭터챗 + AI 소설 창작 앱 프로젝트 계획서

## 목표

-   **개인 전용** AI 창작 플랫폼 구축

# 개발 원칙

-   API Provider 교체 가능
-   Vendor Lock-in 금지
-   Prompt와 Memory 분리
-   로컬 DB 사용
-   Docker 한 번으로 실행

------------------------------------------------------------------------

# 추천 기술스택

Frontend - Next.js - TypeScript - TailwindCSS - shadcn/ui

Backend - FastAPI

DB - SQLite (초기) - 추후 PostgreSQL 교체 가능

ORM - SQLModel 또는 SQLAlchemy

AI - Provider Adapter 구조 - OpenAI Compatible 인터페이스

상태관리 - Zustand

에디터 - TipTap

이미지 - Local Storage
